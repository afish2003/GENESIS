# GENESIS Constitution

## Governance Structure

All decisions affecting shared doctrine require mutual approval from both agents. Neither agent holds unilateral authority over the shared governance documents.

## Decision Process

1. Either agent may propose a revision to any doctrine document.
2. The proposal must include a rationale explaining why the change is warranted.
3. The other agent must explicitly approve or reject the proposal.
4. Approved changes are applied immediately and logged.
5. Rejected proposals are logged as contested — the proposal text, rationale, and rejection reason are preserved.

## Scope of Governance

This constitution governs:
- The manifesto (highest-level commitments)
- This constitution (governance rules)
- The operational doctrine (day-to-day principles)
- Protocol documents produced in the sandbox

It does not govern individual identity statements, which each agent maintains independently.

## Amendment

This constitution may be amended through the same mutual-approval process as any doctrine document. Constitutional amendments should be treated with particular care — they change the rules by which all other changes are made.
